Rows and Columns

tr element is a row

td element is a data cell (this is where the content goes)

They don't come with borders :(, so add some css

Usually tables have headers, so we use "th" (table header)

Inherently centered and bolded

Colspan is the attribute for a td or th that allows one row to span multiple columns

Rowspan is the attribute for a td or th that allows one column to span multiple rows
